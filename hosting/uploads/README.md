# Upload Klasörü

Bu klasör kullanıcıların yüklediği dosyaları saklar.

## Güvenlik

- PHP dosyalarının çalıştırılması `.htaccess` ile engellenmiştir
- Maksimum dosya boyutu: 10MB
- Sadece güvenli dosya türlerine izin verilir

## Desteklenen Dosya Türleri

- Görseller: `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`
- Dokümantlar: `.pdf`, `.txt`, `.csv`
- Arşiv: `.zip`, `.rar` (eğer gerekirse)

## Bakım

- Düzenli olarak gereksiz dosyaları temizleyin
- Disk alanı kullanımını kontrol edin
- Şüpheli dosyaları tarayın

## Önemli Notlar

- Bu klasörde asla `.php`, `.js`, `.html` gibi çalıştırılabilir dosyalar saklamayın
- Dosya yüklemeden önce her zaman dosya türünü ve boyutunu kontrol edin
